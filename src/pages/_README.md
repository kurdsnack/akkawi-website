# src/pages/

Each `.astro` file here is a route. Astro uses file-based routing — the file path matches the URL exactly.

See [CLAUDE.md](../../CLAUDE.md) for the full routing table.

## Naming rules
- Use lowercase, hyphenated slugs (e.g., `solitaire-rings.astro`)
- Collection sub-pages live inside `collections/diamond/` and `collections/gold/`
- No "copy-of-*" or typo slugs — clean names only (fixes the main structural issue with the old Wix site)
